# Boomtown:
An application based on borrowing items people share and also see what others share and have too.

## Usage:
- Raect
- Redux
- Material-UI

## BackEnd: 
- Apollo Server
- Express
- PostgresSQL

### Installation: 
Download the repository and run _npm install_

- Run Server-side:
npm run start:dev --prefix server
- Run Client-side:
npm run start --prefix client
